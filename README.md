# Centergy Group Project Success Simulator

**Reality-Based Predictive Analysis Tool**  
Built from Centergy Group Reality-Based Project Controls + Darwinian Approach

## What It Does
Instantly predicts project success (Green / Yellow / Red) using configurable aspects.  
Especially powerful for **Grant Submissions** (NGOs / Government) while working for any project type.

## Key Features
- Clean, branded UI with Centergy Group logo
- Fully configurable Aspects (add, edit, delete, change weights)
- Dedicated Grant Mode with compliance, outcomes, funding, and alignment aspects
- Responsive advice focused on fixing issues early (Sticky Shuffle, PUF, NOFO compliance, etc.)
- EVM-style visual charts (aspect scores + learning trend)
- Persistent Darwinian learning from actual outcomes
- Easy Markdown export (convert to PDF in one step)
